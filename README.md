## My Setup
    Linux Distro: arch
    Window Manager: i3-gaps
    Bar: i3blocks
    Launcher: Rofi
    Terminal Emulator: terminator
    Shell: zsh
    Terminal Font: System San Francisco Display
    File Manager: nautilus
    Text Editor: nvim
    Music Player: mpd + ncmpcpp
    Browser: firefox
    Video Player: bomi
